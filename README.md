# HR Analytics Power BI Dashboard

## Project Overview
This project analyzes employee data to understand attrition trends, salary distribution, and workforce insights using Power BI.

## Features
- Attrition analysis by Job Role
- Department-wise employee distribution
- Salary vs Experience analysis
- Overtime status analysis
- Education distribution of employees

## Tools Used
- Power BI
- Excel
- Data Visualization

## Dataset
HR employee dataset used for analyzing workforce patterns.
